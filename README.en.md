# jianyi-blog

#### Description
jainyi 博客系统

#### Software Architecture
后端   java语言编写  springboot - mybatis -mysql -shiro

#### 结构

1.  ---db 数据库脚本文件
2.  ---system 博客功能模块

#### 运行

1.  运行 JianyiBlogApplication.java 进行后台的运行  注意配置数据库文件
2.  前端 配置端口ip 则可直接运行 npm run dev
